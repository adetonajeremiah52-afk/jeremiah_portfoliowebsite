# Jeremiah Portfolio Website
A modern, responsive, and interactive portfolio website showcasing my skills, projects, and services as a Software Developer.

## Features

### Navigation
* Sticky Navigation Bar
* Smooth Scrolling Navigation
* Active Section Highlighting
* Mobile Hamburger Menu
* Auto-Close Mobile Menu

### Hero Section
* Professional Introduction
* Animated Typing Effect
* Call-To-Action Button

### About Section
* Personal Introduction
* Statistics Cards
* Responsive Layout

### Projects Section
* Project Showcase Gallery
* Modern Project Cards
* Project Descriptions
* Space for View Project

### Services Section
* Frontend Development
* Backend Development
* Debugging & Problem Solving

### Contact Section
* Contact Information
* Social Media Links
* WhatsApp Integration Ready

### User Experience
* Scroll Reveal Animations
* Hover Effects
* Floating Hero Image
* Back-To-Top Button
* Fully Responsive Design

### Additional Features
* Download CV Button
* Font Awesome Icons
* Mobile-Friendly Navigation
